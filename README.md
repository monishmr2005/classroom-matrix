Classroom Matrix

A lightweight, responsive academic workspace that connects teachers and students for real-time task tracking. Built with pure vanilla frontend technologies and simulated on a localized browser storage data layer.



Features

Dual Dashboards: Custom role-based user interfaces for both Teachers and Students.

Task Engine: Seamless assignment broadcasting, updates, and deletion tracking.

Live Analytics: Visual class progress metrics for instructors and individual status tracking for students.

Theme Sync: Responsive Dark Mode & Light Mode toggle that persists across sessions via localStorage.



Tech Stack

HTML5 / CSS3 (Modern Variables & Grid)

Vanilla JavaScript (ES6+)

Web Storage API (localStorage)
